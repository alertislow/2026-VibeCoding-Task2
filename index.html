關於我

你好！我是 Gleen，一名專注於 React 與現代前端技術棧的開發者。目前我投入於軟體開發與技術教學，擅長將複雜的後端邏輯轉化為直覺、流暢的使用者介面，並熱衷於透過接案挑戰解決多元的商業問題。

我做事的風格始終堅持「溝通先行」，因為我深信清楚的需求對齊是專案成功的基石。我喜歡教學，也喜歡在開發過程中把邏輯講清楚、將程式碼寫得優雅且易於維護，確保每一行程式碼都能為產品創造價值。

  • 卓越的跨部門溝通：能將技術語言轉化為商業語言，確保開發目標與產品方向一致。
  • 現代前端架構設計：擅長使用 React / TypeScript 打造高效、可擴展的組件庫與應用程式。
  • 問題導向的解決力：熱衷於優化網頁效能與使用者體驗，解決開發中的疑難雜症。
